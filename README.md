# 🎬 Netflix Movies Data Analysis

## 📌 Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python to uncover trends in content distribution, genres, countries, ratings, and release years. The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization to gain meaningful insights into Netflix's content library.

## 🎯 Objectives
- Clean and preprocess the Netflix dataset.
- Analyze the distribution of Movies and TV Shows.
- Explore content trends over the years.
- Identify the most common genres and ratings.
- Analyze country-wise content distribution.
- Visualize insights using charts and graphs.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Dataset
The dataset contains information about Netflix Movies and TV Shows, including:
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre

## 📊 Project Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Handling Missing Values
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Business Insights

## 📈 Key Insights
- Distribution of Movies vs TV Shows.
- Most popular genres available on Netflix.
- Country-wise content availability.
- Content added over different years.
- Most common content ratings.
- Release year trends and growth of Netflix's catalog.

## 📷 Visualizations
The project includes visualizations such as:
